# ICS LMS - Community Section (React)

React + Vite implementation of the Community section UI.

## Run

```powershell
cd D:\All Projects\ICS-LMS
npm install
npm run dev
```

Open `http://localhost:5173`

## Structure

- `src/App.jsx` - App entry rendering the Community section
- `src/sections/community/CommunityPage.jsx` - Community section UI and behavior
- `src/sections/community/community.css` - Section styling matching the provided design
